1 行政執行流程（三階段）
01
表單收名單
  • 姓名 + 聯絡方式
  • 會員身分確認
  • 想參加賽事（WEPOWER / 鋼鐵 / 兩場）
  • 想參加組別（推廣 / 菁英）
  • 是否已有隊友
  • 是否需要四季協助組隊
  • 是否參加備賽團練
  • 是否要一起出團
  • 交通 / 住宿 / 餐會需求
  • 是否代表四季出團（核心確認）
02
名單分類整理
  • WEPOWER 推廣組名單
  • WEPOWER 菁英組名單
  • 鋼鐵推廣組名單
  • 鋼鐵菁英組名單
  • 需要組隊名單
  • 只出團不備賽名單
  • 親友同行名單
  • 交通 / 住宿 / 餐會需求名單
  • 非四季出團名單（分開處理）
03
建立群組
  • 4SC 出團總群：所有要跟四季一起出團的人
  • 用途：比賽資訊、集合、交通、住宿、餐會、賽後照片
  • 4SC 備賽團練群：有參加備賽課的人
  • 用途：課程提醒、動作方向、教練公告
  • 兩個群要分開，不要合在一起
2 交通、住宿與餐會安排原則
WEPOWER（台北大直）
  • ·以自行前往或約定集合為主
  • ·四季協助統一集合時間與現場集合點
  • ·原則上不特別安排住宿
  • ·賽後可評估簡單聚餐或拍照
台中鋼鐵大賽（10/17–18）
  • ·依人數評估：自行 / 高鐵集合 / 包車
  • ·包車費用由參與者平均分攤，未達包車人數改高鐵
  • ·住宿：四季推薦區域，大家自行訂房，不代收款
  • ·賽後餐會：四季協助訂位，餐費自行負擔
  • ·臨時取消餐會座位者，需自行承擔相關費用
對外說法（包車）：「交通會依人數評估是否包車，包車費用由參與者平均分攤。未達包車人數，會改成高鐵或自行前往，四季協助安排集合時間。」

對外說法（餐會）：「四季會協助預訂賽後餐廳，餐費由參與者自行負擔。為了方便訂位，會提前統計人數，臨時取消可能需要自行負擔餐廳訂位相關費用。」
3 教練執行重點（要做的事）
  • ✓ 看到適合的會員，主動引導填表單
  • ✓ 會員問組隊，統一導回四季出團流程，不私下亂配
  • ✓ 會員問備賽，導到對應組別與時段
  • ✓ 推廣組以安全、理解、完賽為主
  • ✓ 菁英組以強度、策略、配速、效率為主
  • ✓ 所有規則與動作標準，以官方公告為準
  • ✓ 名單、組隊與出團資訊回到四季統一流程
  • ✓ 讓會員知道為什麼平常訓練跟比賽有關
  • ✓ 有安全疑慮的會員，提醒事先告知或評估
  • ✓ 比賽當天以簡單、清楚、容易執行的提醒為主
✕ 教練不要做的事
  • ✕ 不承諾四季會處理所有交通、住宿與報名費
  • ✕ 不替會員保證名次或完賽結果
  • ✕ 不私下收費開個人備賽小團
  • ✕ 不把代表其他場館的人放進四季出團安排
  • ✕ 不在現場批評會員表現
  • ✕ 不把賽事規則講成四季版本
  • ✕ 不各自開小群，造成資訊分散
  • ✕ 不私下亂配隊，造成行政無法管理
  • ✕ 不批評其他場館、其他隊伍或其他比賽
  • ✕ 不使用「官方合作」「指定訓練中心」等字眼
4 比賽當天現場分工

比賽當天要避免所有事情都集中在 Simon 身上。以下四個角色需事前分配清楚。

出團總召
  • 集合時間確認
  • 集合地點確認
  • 現場公告
  • 臨時狀況協調
  • 統一對外窗口
教練支援
  • 帶領熱身
  • 動作提醒
  • 配速提醒
  • 隊伍分工提醒
  • 賽後簡單照顧
  • 不臨時改太多策略
行政支援
  • 名單確認
  • 聯絡事項
  • 交通確認
  • 餐會確認
  • 群組公告
  • 臨時訊息整理
影像紀錄
  • 出發集合照片
  • 隊伍合照
  • 熱身畫面
  • 比賽動作
  • 隊友加油
  • 完賽表情
  • 賽後餐會
5 對外溝通口徑（建議說法）
我沒有很強，可以參加嗎？
可以先填表單，讓我們看你適合哪個組別。這次重點不是一定要很強才去，而是用比賽當成接下來訓練的目標。我們會協助你評估比較適合的方式。
我不參加團練，可以一起去嗎？
可以。這次會分成備賽團練跟出團同行兩種方式。你如果平常自己練，也可以跟四季一起去現場、一起集合、一起加油、一起拍照。
交通住宿四季會處理嗎？
四季會協助統計需求，也會看人數評估包車、住宿建議與餐會訂位。不過交通、住宿、餐費原則上由參與者自行負擔。
我已經跟別的場館組隊，可以加入四季出團嗎？
如果你這次主要是跟其他場館組隊與行動，就照原本場館安排走會比較清楚。現場遇到我們一樣可以打招呼、互相加油。
我沒有隊友怎麼辦？
先填表單，我們會統整想參加的人，再依照組別、能力、時間安排協助組隊。
備賽課一定要參加整期嗎？
建議一期 4 次一起上，會比較知道動作、流程跟團隊分工。如果時間真的無法配合，也可以單次報名，但前面課程內容需要自己熟悉。

6 這次成功的判斷標準

不只看報名人數，也不只看成績。真正要觀察的十件事：

有多少會員因為這次比賽增加上課頻率
有多少會員第一次願意挑戰外部賽事
有多少人需要四季協助組隊
備賽團練出席率是否穩定
比賽當天是否有明顯的團隊感
賽後是否有會員願意分享心得
是否產生可延伸的照片、影片與文章
是否能帶動下一場出團或下一波訓練方案
教練是否能依照流程引導會員
行政是否能用這套流程複製到下一場活動
內部結論

這套流程，是未來所有出團的範本

→ WEPOWER 是第一場近程出團
→ 台中全球極限體能鋼鐵大賽是正式遠征出團
→ 備賽團練是讓會員有訓練目標
→ 組隊與出團是讓會員產生社群連結
→ 賽後照片、心得與內容，是讓四季品牌被看見
所有夥伴與教練要共同傳達的是：
四季不只提供課程。
四季會帶大家一起準備，一起出發，一起完成挑戰。
